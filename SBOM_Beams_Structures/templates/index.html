SBOM Structural Bill Of Materials

Here are some Of the rules that a drawing should contain:

The List of Keywords which should not be repeated both near text from where it should be extracted and in the diagram - ISMB,ISMC,MKD,PLT,THK,SHEET NO, REV NO,

The Green color Border should be of Thickness 1.20 mm.

One beam should contain only one MKD keyword.

The MKD keyword and values should be written as

 a. Single Value - MKD. B12

 b. Multiple Values - MKD. B12,B13,B14

 c. TO Keyword Values - MKD. B12 TO B19

 d. Multi-line Values - MKD. B10,B11,B12,B13,

              B14,B15,B16,B17

Write Full abbrevations ISMB, ISMC not MB, MC

Gap between ISMB 300 X 6000 Lg. Gaps between each Of them should be present as shown.

The Sheet Number and Revision Number should be present as follows:

SHEET NO. 1 OF 2

REV NO. R1

PLT to THK keyword present on first line - Quantity present on next line

PLT 150 X 200 X 8 THK

- 1 NOs

ISA to THK keyword present on first line - Length and Quantity present on next line

ISA 100 X 100 X 10 THK

350 Lg - 2 NOs

# Structural-Drawing-BOM-Automation
Shrijee Bill of Materials (S-BOM) is an OCR-based automation system that extracts material details from structural engineering drawings and generates a structured Bill of Materials in spreadsheet format, improving efficiency in BOM generation and construction and manufacturing workflows.
